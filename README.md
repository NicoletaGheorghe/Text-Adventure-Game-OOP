Run Locally:

To run the project on your machine:

1. Clone the repository:

https://github.com/NicoletaGheorghe/Text-Adventure-Game-OOP.git

2. Navigate into the folder:

cd Text-Adventure-Game-OOP

3. Open index.html in your browser

You can simply double-click the index.html file 
OR use a live server (e.g., VSCode Live Server extension) for a better experience.
